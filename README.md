## Description
_Fight-Games_
My first coding project

A technical demonstration of collision detection, gravity physics, and character state management using Object-Oriented Programming in JavaScript.

2D Browser-Based Fighting Game | JavaScript, HTML, CSS
- Developed a fully functional 2D combat engine using Vanilla JavaScript and HTML Canvas for high-performance rendering.
- Engineered custom collision detection and gravity physics to manage player interactions and environment boundaries.
- Implemented Object-Oriented Programming (OOP) to manage character states, including health, movement, and unique attack animations.
- Optimized game loop performance to ensure consistent 60FPS gameplay across different browsers.
